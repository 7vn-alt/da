Steps to Create Dashboard (Sample Superstore) 1. Data Cleaning a) Apply filters to all columns b) Check for anomalies such as blanks, incorrect data c) Correct anomalies manually 2. For each goal, perform analysis a) Insert Pivot Table in new sheet b) Drag fields on which analysis has to be done respective to tab of pivot table feild c) Remove Grand Total: Select pivot table→Checl Design Robbon → Grand Total → Off d)Select option ribbon→pivot chart oprion→select chart type→OK e)You will get desired chart 3. Create Charts Follow step 2 to create all the charts in a new sheet, name it as excel dashboard. 4. Add Slicers Goto Pivot Chart Tools Ribbon → Analyse option → Insert Slicer → Select column to insert. Analyze → Insert Slicer Choose fields for slicers 5. Connect Slicers Slicer Tools → Options → Pivot Table Connections Select all pivot tables to connect 6. Dashboard will be interactive , selected data from slicer will dynamically update to display the options
Execution
1. Data Cleaning 1. Select the entire dataset (Ctrl + A). 2. Go to Data → Filter. 3. Open each filter dropdown and check for: • Blank cells • Wrong or inconsistent values • Spelling or category errors 4. Fix all issues manually.
2. Create Pivot Tables for Each Goal 1. Select data → Insert → PivotTable → New Worksheet → OK. 2. In PivotTable Fields panel: • Drag field to Rows (e.g., Category/Region/Segment). • Drag Sales/Profit/Quantity to Values. 3. Turn off Grand Totals: • Click pivot → Design → Grand Totals → Off.
3. Create Charts 1. Click inside the pivot table. 2. Insert → PivotChart. 3. Choose chart type based on the goal: • Doughnut → Segment-wise profit • Pie → Category quantity • Bar/Column → Region-wise sales & profit 4. Click OK. 5. Format the chart (title/colors). 6. Copy chart → Paste into a new sheet named “Excel Dashboard”.
4. Add Slicers 1. Select any pivot table or chart. 2. PivotTable Analyze → Insert Slicer. 3. Choose slicer fields (Region / Category / Segment).
5. Connect All Slicers 1. Click a slicer. 2. Slicer Tools → Options. 3. PivotTable Connections. 4. Check all pivot tables that should respond to the slicer. 5. Repeat for each slicer.

import random

a = int(input("enter no of trails: "))
b = int(input("each input how many rolls: "))
c = int(input("enter rollup value: "))
d = 0

for i in range(a):
    for j in range(b):
        result = random.randint(1, 6)
        print(result)
        if result == c:
            d += 1
    print("---")

total = a + b
print(f'number of times {c} appeared: {d}')
print("probability =", d/total)
